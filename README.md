# Quick Commerce Business Analysis using SQL

## Project Overview
This project analyzes quick commerce order data to understand business performance, delivery efficiency, customer satisfaction, discount strategies, and product demand.

## Tools Used
SQL (MySQL)

## Dataset
The dataset contains approximately 1 million orders across multiple cities and companies.

## Analysis Performed
- KPI Summary
- Revenue and Sales Analysis
- Delivery Performance Analysis
- Customer Ratings Analysis
- Discount Impact Analysis
- Order Behavior Analysis

## Key Insights
- The platform generated about ₹57 crore in revenue.
- Average order value is ₹572.
- 97% of orders are delivered within 30 minutes.
- Gurgaon generates the highest revenue.
- Hyderabad has the highest order volume.
- Dairy products generate the highest demand.
